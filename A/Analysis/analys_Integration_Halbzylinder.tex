\Aufgabe[e]{Volumen in Zylinderkoordinaten}
{
Berechnen Sie das Volumen des Körpers

\[
V = \left\{ (x, y, z) \in \mathbb{R}^3 \ \middle| \
\begin{array}{l}
x^2 + y^2 \leq R^2, \,
x \geq 0, \,
0 \leq z \leq 2x + y + 1
\end{array}
\right\}
\]

Bestimmen Sie das Volumen dieses Bereichs unter Verwendung von Zylinderkoordinaten.
}

\Loesung{
Zylinderkoordinaten:
\[
x = r \cos \theta, \quad y = r \sin \theta, \quad z = z
\]

Die obere Begrenzungsebene ist

\[
z = 2x + y + 1 = 2r \cos \theta + r \sin \theta + 1 = r(2\cos \theta + \sin \theta) + 1
\]

Das Volumenelement in Zylinderkoordinaten ist:

\[
dV = r \, dz \, dr \, d\theta
\]

Da es sich um einen Halbzylinder mit $x \geq 0$ handelt, gelten folgende Grenzen:

\begin{align*}
0 \leq r \leq R \\
-\frac{\pi}{2} \leq \theta \leq \frac{\pi}{2} \\
0 \leq z \leq r(2\cos \theta + \sin \theta) + 1
\end{align*}

Das Volumen ergibt sich zu:

\[
V = \int_{-\frac{\pi}{2}}^{\frac{\pi}{2}} \int_0^R \int_0^{r(2\cos\theta + \sin\theta) + 1} r \, dz \, dr \, d\theta
\]

Integration über $z$:

\[
\int_0^{r(2\cos\theta + \sin\theta) + 1} r \, dz = r \cdot \left[ r(2\cos\theta + \sin\theta) + 1 \right]
\]

Das ergibt:

\[
V = \int_{-\frac{\pi}{2}}^{\frac{\pi}{2}} \int_0^R \left[ r^2(2\cos\theta + \sin\theta) + r \right] \, dr \, d\theta
\]

\[
\int_0^R r^2(2\cos\theta + \sin\theta) \, dr = (2\cos\theta + \sin\theta) \cdot \frac{R^3}{3}
\]

\[
\int_0^R r \, dr = \frac{R^2}{2}
\]

Damit:

\[
V = \int_{-\frac{\pi}{2}}^{\frac{\pi}{2}} \left[ \frac{R^3}{3}(2\cos\theta + \sin\theta) + \frac{R^2}{2} \right] \, d\theta
\]

Bekannte Integrale:

\[
\int_{-\frac{\pi}{2}}^{\frac{\pi}{2}} \cos\theta \, d\theta = 2, \quad
\int_{-\frac{\pi}{2}}^{\frac{\pi}{2}} \sin\theta \, d\theta = 0, \quad
\int_{-\frac{\pi}{2}}^{\frac{\pi}{2}} d\theta = \pi
\]

Somit:

\[
V = \frac{R^3}{3} \cdot 4 + \frac{R^2}{2} \cdot \pi = \frac{4R^3}{3} + \frac{\pi R^2}{2}
\]

Das gesuchte Volumen beträgt:

\[
\boxed{V = \frac{4R^3}{3} + \frac{\pi R^2}{2}}
\]
}
